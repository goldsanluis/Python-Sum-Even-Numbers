# Sum of Even Numbers

A Python function that iterates through a list of integers
and returns the total sum of only the even numbers.

## What It Does
- Skips all odd numbers
- Returns 0 if no even numbers are present
- Raises a TypeError if input is not a list

## Sample Output
sum_even_numbers([1, 2, 3, 4, 5, 6])  # 12
sum_even_numbers([1, 3, 5, 7])         # 0
sum_even_numbers([2, 4, 6, 8])         # 20

## Requirements
- Python 3.x
